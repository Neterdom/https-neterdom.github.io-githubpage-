2019-11-27

En Aplicaciones Web aprendemos a hacer páginas web desde cero, usando HTML para construir la estructura y CSS para que se vean bonitas. Hacemos cosas como formularios, menús, botones que cambian al pasar el ratón o diseños que se adaptan al móvil. A veces usamos herramientas como Bootstrap para ahorrar tiempo. La verdad es que es bastante entretenido porque ves lo que haces en el navegador y puedes ser creativo mientras aprendes algo muy útil.
